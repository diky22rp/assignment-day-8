# Diky Rizki Pratama - Kesimpulan dan Pandangan dalam Penggunaan GitHub saat Membangun Proyek dalam Tim

GitHub memberikan kemudahan dalam kerja sama tim karena alur kerjanya yang terstruktur, mulai dari pembuatan branch, commit, pull request, hingga proses review yang ada pada git. Dengan menggunakan sistem ini, setiap anggota tim dapat mengerjakan bagiannya masing-masing tanpa saling mengganggu satu sama lain, sekaligus memudahkan tim untuk menelusuri perubahan yang terjadi di dalam proyek.

Melalui fitur pull request, leader tim bisa melakukan pengecekan dan diskusi terlebih dahulu dengan anggkota timnya sebelum kode digabungkan ke branch utama. Cara ini menjaga kualitas hasil kerja code proyek dan memastikan konsistensi antar anggota tim.

Secara pribadi, saya melihat GitHub bukan sekedar tempat menyimpan kode, tetapi juga sarana kolaborasi yang membangun kebiasaan kerja yang terstruktur, terbuka, dan profesional di dalam sebuah tim proyek pengembangan aplikasi.
